Get Started in Minutes

Simple installation process with our one-click web installer

1

Download & Extract

Download larpress and extract it to your web server directory.

2

Run Web Installer

Navigate to your domain and follow the simple web installer wizard.

3

Configure & Launch

Set up your database, admin credentials, and you're ready to go!

System Requirements

  • PHP 8.1 or higher
  • MySQL 5.7+ or PostgreSQL 10+
  • Composer 2.x
  • Node.js 16+ & NPM

Quick Start

terminal
# Clone the repository
git clone https://github.com/Pacific-Softwares/larpress-demo.git

# Install dependencies
composer install
npm install && npm run build

# Run web installer
php artisan serve

# Visit http://localhost:8000/install
